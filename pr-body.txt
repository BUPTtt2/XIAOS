## 本次变更

1. 初始化 Next.js 15 项目配置（TypeScript、Tailwind CSS）
2. 创建剧本 YAML Schema 完整定义（`schema/script-schema.yaml`）
3. 编写 Schema 设计文档，包含结构设计原因（`schema/schema-design.md`）
4. 定义 TypeScript 类型（`lib/types.ts`）
5. 初始化 Prisma 数据库 Schema
6. 创建基础首页页面
7. 补充项目需求文档、方案选择文档和系统骨架文档

## 交付文件
- `package.json` - 项目依赖配置
- `tsconfig.json` - TypeScript 配置
- `tailwind.config.ts` - Tailwind 配置
- `schema/script-schema.yaml` - YAML Schema 定义
- `schema/schema-design.md` - Schema 设计文档
- `lib/types.ts` - TypeScript 类型
- `prisma/schema.prisma` - 数据库 Schema
- `docs/` - 完整项目文档

## 验证点
- Schema 完整包含剧本所有元素（时间、场景、人物、对话、动作、旁白、情感音调）
- 设计文档清晰说明每一部分设计原因
- 项目配置完整，本地可运行
